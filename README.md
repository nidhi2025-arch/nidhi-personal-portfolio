# Nidhi Portfolio Website

Personal portfolio landing page for Nidhi, built with HTML, CSS, and vanilla JavaScript.

## Highlights

- Responsive, mobile-first layout
- Dark/light theme toggle
- Sticky navigation with active section tracking
- Animated reveal effects
- Download resume button
- Contact form that opens the default email client

## Quick Notes

- Update `nidhi@example.com` in `index.html` and `main.js` with your real email.
- Replace the placeholder social links with your LinkedIn and GitHub profiles.
- Swap in your own portrait later if you want a personal photo in the hero area.
